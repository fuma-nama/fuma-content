---
title: Content Studio
description: Experimental support for local content editor.
---

<Callout type="warn" title="Experimental">
  The API is unstable and may have potential performance & UX issues.
</Callout>

## Installation

```npm
npm i @fuma-content/studio -D
```

Start the studio:

```npm
npm content-studio
```

It will start the studio locally.

<Callout type="warn" title="Local Only">
  It only supports local usage at the moment, there is not auth system implemented yet.
</Callout>
